# Electric Circuit Experiment

An interactive virtual laboratory for learning about electric circuits, circuit symbols, current flow, and voltage measurements.

## Features

- **Interactive Circuit Building**: Drag and drop components to build circuits
- **Circuit Components**: 
  - Battery (voltage source)
  - Resistor
  - LED (Light Emitting Diode)
  - Switch
  - Ammeter (current measurement)
  - Voltmeter (voltage measurement)
- **Real-time Measurements**: 
  - Current (I) in Amperes
  - Voltage (V) in Volts
  - Resistance (R) in Ohms
  - Power (P) in Watts
- **Visualizations**:
  - Conventional current flow (red arrows, + to -)
  - Electron flow (blue dots, - to +)
  - Circuit connections
- **Ohm's Law**: Real-time calculation and display of V = I × R
- **Data Logging**: Record and analyze circuit measurements

## Educational Objectives

- Understand circuit symbols for common electronic components
- Learn about current, voltage, and resistance relationships
- Observe the difference between conventional current and electron flow
- Practice using ammeters and voltmeters
- Apply Ohm's Law in practical scenarios

## Usage

1. Drag components from the palette onto the circuit canvas
2. Adjust voltage and resistance using the sliders
3. Observe real-time current and voltage measurements
4. Watch the animated current and electron flow
5. Use preset circuits to explore different configurations
6. Record data points to analyze relationships

## Technical Details

- Built with HTML5, CSS3, and vanilla JavaScript
- SVG for circuit connections and flow visualization
- Real-time calculations using Ohm's Law
- Drag-and-drop interface for component placement

## Concepts Covered

- **Ohm's Law**: V = I × R
- **Power Formula**: P = V × I
- **Conventional Current**: Flow from positive to negative terminal
- **Electron Flow**: Actual movement of electrons from negative to positive
- **Circuit Symbols**: Standard symbols for electronic components
- **Series and Parallel Circuits**: Different circuit configurations
