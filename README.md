# Keith's Web Portal

這是一個個人網頁入口，提供快速訪問各個專案與頁面。

## 專案列表

1. **Profile** - 個人簡介與履歷（未來會整理）
2. **PrimaryChi** - 語言大富翁遊戲專案，連接到 [keithsflau/primarychi](https://github.com/keithsflau/primarychi)
3. **SecondarySci** - 科學相關專案（未來會整理）

## 使用方式

直接在瀏覽器中打開 `index.html` 或部署到 GitHub Pages。

### GitHub Pages 部署

1. 將此倉庫推送到 GitHub
2. 在倉庫設置中啟用 GitHub Pages
3. 選擇 main branch 作為來源
4. 訪問 `https://keithsflau.github.io/profile/`
