===============================================
    快速開始指南
===============================================

📤 上傳到 GitHub 並檢查鏈接

【最簡單方法】
─────────────────────────────────────────
直接雙擊運行：DO_IT_ALL.bat

這個文件會自動完成：
  ✓ 添加所有更改
  ✓ 提交到 git
  ✓ 推送到 GitHub  
  ✓ 檢查死鏈接


【其他方法】
─────────────────────────────────────────

方法 1：分開執行
  1. 雙擊：upload_to_github.bat  （上傳）
  2. 雙擊：check_broken_links.bat （檢查鏈接）

方法 2：使用 Python
  雙擊：upload_and_check.py


【如果腳本無法運行，手動執行】
─────────────────────────────────────────
打開命令提示符（CMD），在項目根目錄執行：

git add -A
git commit -m "更新所有 footer 為包含聖經經文的格式 (Jeremiah 10:12)"
git push origin main
python check_links_simple.py


【已準備好的文件】
─────────────────────────────────────────
✓ DO_IT_ALL.bat              - 一鍵執行所有操作
✓ upload_to_github.bat       - 只上傳到 GitHub
✓ check_broken_links.bat     - 只檢查鏈接
✓ upload_and_check.py        - Python 版本（所有操作）
✓ check_links_simple.py      - 鏈接檢查腳本


【需要幫助？】
─────────────────────────────────────────
查看 README_LINK_CHECK.md 獲取詳細說明
