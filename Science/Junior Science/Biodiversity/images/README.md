# Biodiversity Images

This folder contains images for the Biodiversity page.

## Required Images

Please add the following images to this folder:

1. **spoonbill.jpg** - Black-faced Spoonbill photo
   - Recommended: High-quality photo of a Black-faced Spoonbill bird
   - Dimensions: 800x600px or larger
   - Format: JPG

2. **dolphin.jpg** - Chinese White Dolphin photo
   - Recommended: High-quality photo of a Chinese White Dolphin (Pink Dolphin)
   - Dimensions: 800x600px or larger
   - Format: JPG

3. **frog.jpg** - Romer's Tree Frog photo
   - Recommended: High-quality photo of Romer's Tree Frog
   - Dimensions: 800x600px or larger
   - Format: JPG

## Image Sources

You can find suitable images from:
- Unsplash (https://unsplash.com)
- Pexels (https://www.pexels.com)
- Pixabay (https://pixabay.com)
- Or use your own photos

## Fallback

If images are not found in this folder, the page will automatically use Unsplash images as fallback.
